## **1. O que é um array em programação?**
É uma lista que armazena os elementos de um mesmo tipo de dado, como números, strings ou objetos.

---

## **2. Por que é melhor usar um array do que criar várias variáveis separadas, como aluno1, aluno2 e aluno3?**
Usar um array é melhor, porque ele permite armazenar múltiplos valores em uma só váriavel.

---

## **3. O que é índice em um array?**
O índice é a posição de cada elemento dentro do array, começando do 0.

---

## **4. Por que o primeiro elemento de um array fica na posição 0?**
Porque a contagem de índices na programação começa do 0.

---

## **5. Para que serve o .length?**
Serve para saber o tamanho do array, ou seja, quantos elementos ele possui.

## **6. Como acessamos o primeiro elemento de um array chamado alunos?**
alunos[0]

---

## **7. Como podemos alterar um valor dentro de um array?**
alunos[índice] = novoValor

---

## **8. Qual a relação entre arrays e laços de repetição?**
Os laços de repetição, como for e while, são usados para percorrer os elementos de um array.

---

## **9. Qual a diferença entre percorrer um array com for e apenas mostrar um item específico?**
Percorrer um array com for permite acessar todos os elementos do array, enquanto mostrar um item específico acessa apenas o elemento desejado através do seu índice.

---


## **10. O que é um acumulador? Dê um exemplo de uso com notas ou valores.**
Um acumulador é uma variável que armazena a soma ou o total de valores ao longo de um processo.

numero 1 = 10;
numero 2 = 20;
acumulador = (numero 1 + numero 2);
acumulador = 30;

## **11. Para que servem operadores aritméticos como +, -, * e / dentro de um programa?**
Eles servem para realizar operações matemáticas básicas, como adição, subtração, multiplicação e divisão, permitindo manipular valores numéricos dentro do programa.

---

## **12. Para que servem operadores lógicos como && e ||?**
Eles servem para combinar condições em expressões booleanas. O operador && retorna verdadeiro se ambas as condições forem verdadeiras, enquanto o operador || retorna verdadeiro se pelo menos uma das condições for verdadeira.

---

## **13. Qual a diferença entre if/else e switch?**
A diferença é que o if/else é usado para tomar decisões com base em condições booleanas, enquanto o switch é usado para comparar um valor com múltiplos casos possíveis, tornando o código mais organizado quando há várias opções.

---

## **14. O que os métodos push e pop fazem em um array?**
O método push adiciona um ou mais elementos ao final de um array, enquanto o método pop remove o último elemento do array.

---

## **15. O que é uma matriz e como ela se relaciona com arrays?**
Uma matriz é uma estrutura de dados bidimensional que pode ser vista como um array de arrays. Ela permite armazenar elementos em linhas e colunas, sendo útil para representar tabelas ou grades de dados.
