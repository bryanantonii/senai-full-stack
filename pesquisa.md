## **Perguntas**

**1) O que é programação?**
---
É o processo de criar um conjunto de instruções organizadas e lógicas para que um computador execute tarefas especificas.

---
**2) O que é um algoritmo?**
---
É uma sequência de instruções lógicas, finitas e bem definidas, criada para resolver um problema ou executar uma tarefa bem definida.

---

**3)Qual a diferença entre algoritmo e linguagem de programação?**
---
O algoritmo é a lógica ou "Passo a Passo" para resolver um problema. Uma linguagem de programação é a ferrenta usada para escrever esse algoritmo de forma que o computador entenda.

---

**4)O que é JavaScript?**
---
Uma linguagem de programação utilizada para adicionar dinamismo, interatividade, inteligência a paginas web.

---

**5)Onde o JavaScript pode ser utilizado atualmente?**
---
No desenvolvimento de aplicações web(front-end e back-end), aplicações mobile, sistemas desktop, jogos e ate mesmo internet das coisas.

---

**6)O que é o Node.js?**
---
É um ambiente de execução(run time)de código aberto que permite executar códigos de javascript diretamente do computador/ou servidor, fora do navegador web.

---

**7)Qual é a função do Node.js no desenvolvimento com JavaScript?**
---
Permitir que os desenvolvedores executem códigos do javascript fora do navegador.

---

**8)O que é o VS CODE?**
---
É um editor de código fonte gratuito e de código aberto desenvolvido pela microsoft

---

**9)Qual a diferença entre um editor de texto comum e um de código?**
---
Um editor comum como bloco de notas, serve apenas para escrever texto puro. Já o editor de código é especializado para programação, oferecendo recursos como destaques e cores, autocompletar, identificação de erros e integração com ferramentas de versionamento

---

**10)O que é o terminal?**
---
O Terminal é um aplicativo moderno e avaçado criado pela Microsoft para interagir com o sistema operacional por meio de linhas de comando.

---
**11) Para que serve o comando node arquivo.js?**
---
O Comando node arquivo.js serve para executar um arquivo de código JavaScript diretamente no seu computador ou servidor, fora do ambiente de um navegador web.

---
**12) O que é uma variavel?**
---
É um elemento ou simbolo que representa um valor, ou caracteristica que pode mudar ou ser alterado.

---

**13)Qual a diferença entre Let e Const?**
---
É a capacidade de reatribuição, variáveis declaradas com let, podem rer seus valores alterados posteriormente, enquanto declaras comn const são constantes e não podem receber um novo valor após a inicialização

---

**14)O que são tipos de dados? Cite exemplos**
---
São classificações que indicam ao computador qual é a natureza de um valor. Eles definem o espaço que esse dado ocupará na memória e quais operações podem ser realizadas com ele.

---

**15)Qual a principal diferença entre JavaScript e TypeScript?**
---
o JavaScript é dinamicamente tipado (não exige declaração de tipos), enquanto o TypeScript é estaticamente tipado (permite definir e validar os tipos de dados).
