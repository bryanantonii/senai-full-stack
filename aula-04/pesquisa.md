## **1. O que e uma estrutura de repeticao?**
É um comando que permite executar um bloco de comando repetida vezes até que o número desejado ou a condição específica seja alcançada.

---

## **2. Para que serve o laco for?**
É uma estrutura de repetição usada na programação para executar um bloco de código repetidas vezes.

---

## **3. Para que serve o laco while?**

 Serve para executar um bloco de código repetidamente enquanto uma condição específica for verdadeira
 
 ---

## **4. Para que serve o laco do-while?**
Serve para repetir um bloco de código pelo menos uma vez, garantindo que a verificação da condição ocorra apenas após a primeira execução.

---

## **5. Qual e a diferenca entre for, while e do-while?**
O for é usado quando você sabe exatamente quantas vezes o código deve repetir. O while executa um bloco de código repetidamente enquanto uma condição for verdadeira. O do-while executa o bloco pelo menos uma vez e depois continua repetindo enquanto a condição for verdadeira, com a verificação ocorrendo após a execução.
