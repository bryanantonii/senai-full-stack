## ***1. O que é uma função em programação?**
 Uma função é um bloco de código nomeado e reutilizável, que executa uma tarefa específica.


## ***2. Qual é a principal vantagem de usar funções em vez de repetir o mesmo código várias vezes?**
A principal vantagem é a organização do código, pois com a função você pode chamá-la quantas vezes for preciso sem repetir o código.

## ***3. O que são parâmetros de uma função?** 
Os parâmetros são variáveis declaradas dentro dos parênteses da função.

## ***4. Qual a diferença entre parâmetro e argumento?** 
O parâmetro é a variável declarada dentro na definição de uma função, enquanto o argumento é o valor real que você passa para essa função ao executá-la.

## ***5. Para que serve a palavra-chave return?** 
A palavra chave return serve para encerrar a execução da função e retornar o valor para quem a chamou




## ***6. O que acontece quando uma função não possui return?** 
A função não funciona, sem o return o valor esperado na função não é exibido.

## ***7. O que significa modularizar um código?** 
Modularização é a prática de dividir um programa em módulos ou funções independentes, cada um responsável por uma tarefa específica.

## ***8. Por que funções pequenas e bem nomeadas facilitam a manutenção do sistema?** 
Pois é fácil de identificá-las e entender do que as mesmas se tratam, invés de repetir o mesmo código várias vezes.

## ***9. Em TypeScript, por que é útil informar o tipo dos parâmetros e do retorno?** 
Pois a linguagem de programação TypeScript é fortemente tipada, se não informar o tipo dos parâmetros e do retorno o código não funciona.
