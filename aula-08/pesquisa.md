## ***1. O que é um objeto em programação?**
Um objeto é uma estrutura que representa uma entidade do mundo real, contendo propriedades e métodos.

## ***2. O que é uma propriedade de um objeto? Cite um exemplo.**
Uma propriedade é uma característica ou atributo de um objeto. Exemplo: em um objeto "carro", a propriedade "cor" pode ter o valor "vermelho".

## ***3. O que é um método? Qual a diferença entre método e propriedade?**
Um método é uma função associada a um objeto que define um comportamento. A diferença é que a propriedade armazena dados, enquanto o método executa ações.

## ***4. O que é uma classe?**
Uma classe é um modelo ou molde para criar objetos, definindo suas propriedades e métodos.

## ***5. Qual a diferença entre classe e objeto?**
A classe é o modelo, enquanto o objeto é uma instância concreta desse modelo.

## ***6. Para que serve o constructor dentro de uma classe?**
O constructor é um método especial usado para inicializar objetos criados a partir de uma classe.

## ***7. O que significa a palavra this dentro de uma classe?**
A palavra-chave this refere-se à instância atual da classe, permitindo acessar suas propriedades e métodos.

## ***8. O que a palavra new faz ao criar um objeto?**
A palavra-chave new cria uma nova instância de uma classe, chamando o constructor para inicializar o objeto.

## ***9. Em TypeScript, por que usamos tipos como string, number e boolean?**
Usamos tipos para garantir que as variáveis e propriedades tenham valores esperados, ajudando a evitar erros e melhorar a legibilidade do código.
